Ayo Cari Jalan Terdekat Menuju Tujuan Anda!

About

Aplikasi ini adalah aplikasi yang memanfaatkan algoritma A* untuk mencari jalan terdekat dari lokasi anda menuju lokasi tujuan.

‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎‎Cara Penggunaan :

‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎1. Pilih pilihan kota yang tersedia pada combobox

‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎2. Klik 'Pilih Kota'

‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎3. Klik tempat-tempat yang dapat dituju

‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎4. Masukkan Lokasi Anda

‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎5. Masukkan Lokasi Tujuan

‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎6. Pilih pilihan kota

‏‏‎ ‎‏‏‎ ‎‏‏‎ ‎7. Klik tombol search

Fitur :

~ Visualisasi Map API

~ Click Node dan Path untuk mendapatkan informasi lanjut

~ Visualisai Warna Node dan Path yang memudahkan pembacaan map

~ Alert Exception jika salah dalam menginput node/tidak ditemukan jalan/tidak memilih kota

~ Interactive GUI Website :

1. Link Navigation bar

2. Dinamic Button

3. Combobox map & Round Form Input
